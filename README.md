# 📌 Simple Calculator in C

online gdb link : https://onlinegdb.com/k2YYkl3In

## 📖 Description

This program is a **basic calculator** written in the C programming language. It performs simple arithmetic operations like addition, subtraction, multiplication, division, and modulus using user-defined functions.

---

## ⚙️ Features

* Performs **Addition (+)**
* Performs **Subtraction (-)**
* Performs **Multiplication (*)**
* Performs **Division (/)**
* Performs **Modulus (%)**
* Uses **functions** for each operation
* Uses **switch-case** for menu selection

---

## 🧠 Concepts Used

* Functions in C
* Switch-case statement
* User input using `scanf()`
* Basic arithmetic operations

---

## ▶️ How It Works

1. The user enters two numbers.
2. The program displays a menu of operations.
3. The user selects an operation (1–5).
4. Based on the choice, the program calls the respective function.
5. The result is displayed on the screen.

---

## 📋 Menu Options

* Press 1 → Addition
* Press 2 → Subtraction
* Press 3 → Multiplication
* Press 4 → Division
* Press 5 → Modulus
* Press 0 → Exit

---

## ⚠️ Limitations

* The program runs only once (no loop).
* Division by zero is not handled.
* Works only with integer values.
